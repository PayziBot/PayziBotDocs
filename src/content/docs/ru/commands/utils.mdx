### Прогноз погоды

> Команда - `/weather`
>
> Аргументы - `город*`

Просмотр погоды на 3 дня в указанном городе

### Инфоормация о Github-аккаунте

> Команда - `/github`
>
> Аргументы - `ник*`

Просмотр информации об аккаунте или организации на Github

### Переводчик

> Команда - `/translate`
>
> Аргументы - `язык*`, `текст*`

Переводит указанный текст на выбранный язык.

Доступные для выбора языки: `Английский`, `Русский`, `Украинский`, `Немецкий`,
`Японский`, `Китайский`, `Португальский`, `Казахский`,
`Турецкий`, `Французский`, `Арабский`, `Испанский`

### Информация о Minecraft-сервере

> Команда - `/minecraft server`
>
> Аргументы - `адрес*`

Выводит информацию о Minecraft-сервере, включая версию игры, MOTD, количество игроков и по возможности их юзернеймы

Пример использования: `/minecraft server адрес:play.pepeland.net`

### Информация о Minecraft-аккаунте

> Команда - `/minecraft player`
>
> Аргументы - `ник*`

Выводит UUID, скин и плащ игрока Minecraft. Поддерживаются только ники, имеющие лицензию

Пример использования: `/minecraft player ник:Notch`

### Информация о Discord-сервере

> Команда - `/serverinfo`

Выводит информацию о сервере, на которой выполнена Команда

### Информация о пользователе

> Команда - `/userinfo`
>
> Аргументы - `пользователь`

Выводит информацию о пользователе. Если пользователь не указан, отобразится информация пользователя, выполнившего команду

### Курс валют

> Команда - `/exchange`
>
> Аргументы - `из*`, `в*`, `количество`

Переводит одну валюту в другую

К примеру, для просмотра курса доллара, введите:
`/exchange из:USD в:RUB`

### Аватар пользователя

> Команда - `/avatar`
>
> Аргументы - `пользователь`

Выводит аватар пользователя. Если пользователь не указан, отобразится аватар пользователя, выполнившего команду
